The connected Core2 or CoreS3 is detected automatically.

Flashing resets your saved settings whether or not Erase device is checked. Leaving it unchecked is faster because it skips an additional full-device erase.

The standard remote build uses an external battery. The recommended 500 mA option is compatible with the batteries documented by this project when charged at room temperature. Select 1000 mA only when the installed battery is explicitly rated for a charge current of at least 1000 mA, such as the EEMB LP103454. Retaining a small stock M5 battery is not a supported remote configuration.

Use a browser that supports Web Serial to install firmware. Web Serial requires HTTPS or a page served from localhost.
