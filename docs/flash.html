Web Flasher

Loading firmware release information...

The flasher automatically detects whether the connected OSSM M5 Remote is Core2-based or CoreS3-based.

Flashing resets your saved settings whether or not Erase device is checked during install. Leaving it unchecked is faster because it skips an additional full-device erase.

The standard remote build uses an external battery. Use the recommended 500 mA option unless the installed battery is explicitly rated for charging at least 1000 mA. The documented EEMB LP103454 supports the 1000 mA option. Slower charge rates require building from source.

Remote not listed? Core2-based remotes may require a USB driver. Install the CP210x or CH9102 driver for your operating system from the official M5Stack downloads. Which driver you need depends on the Core2 hardware revision. If unsure, install both. CoreS3-based remotes should not require an additional driver.

  1. Connect the remote using a USB data cable.
  2. Select the battery charge current above.
  3. Click Install firmware.
  4. Select the remote from the browser's serial-device list.
  5. In the flasher dialog, confirm that you want to install the firmware.
  6. Optionally enable Erase device, then click Next.
  7. Wait for flashing to finish. The remote will restart automatically.
Use a current desktop version of Firefox, Chrome, or Edge to install firmware. Web Serial requires HTTPS or a page served from localhost.
